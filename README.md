# RenderRealms

RenderRealms é um projeto de estudo focado em **boas práticas de desenvolvimento backend e arquitetura de APIs modernas**, desenvolvido ao longo do **curso.dev**.

O objetivo do projeto é construir uma aplicação real aplicando conceitos importantes do desenvolvimento profissional, como:

- **Testes automatizados (Jest)**
- **TDD (Test Driven Development)**
- **BDD para organização de testes**
- **Controle de erros e exceções**
- **Criação de erros customizados**
- **Arquitetura de API REST**
- **Controle de métodos HTTP**
- **Migrations de banco de dados**
- **Observabilidade e health checks**
- **Boas práticas de organização de código**

A aplicação possui endpoints como `/api/v1/status`, responsáveis por verificar a saúde do sistema e do banco de dados, além de rotas para gerenciamento de migrations.

O projeto também explora ferramentas modernas do ecossistema JavaScript, como:

- **Next.js**
- **next-connect**
- **node-pg-migrate**
- **Jest**
- **PostgreSQL**

Mais do que apenas implementar funcionalidades, o foco do RenderRealms é **entender profundamente como sistemas reais são construídos**, desde a infraestrutura até o tratamento correto de erros e a garantia de qualidade por meio de testes automatizados.
